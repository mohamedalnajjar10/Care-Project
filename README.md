# Care-Project

Backend API built with **NestJS**.

## Requirements

- Node.js (LTS recommended)
- npm

## Project setup

```bash
npm install

# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod

# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
